# From Network Engineer to Network Automation

This repository documents my journey from traditional network engineering 
to network automation.

I come from a hands-on networking background, working with switching, routing, 
firewalls, and enterprise network troubleshooting. 

Recently, I started transitioning into automation — learning Python, 
Netmiko, and building lab environments in EVE-NG to automate network configuration.

This repository is not a polished tutorial.

It is a real engineering log.

It records:
- Lab topology design
- Network configuration experiments
- Python automation scripts
- Troubleshooting cases
- Mistakes and lessons learned
